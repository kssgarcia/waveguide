Special Functions 
-----------------

ber_bei_0
---------
Kelvin functions ber_0 and bei_0

bes_I01K01
----------
Modified Bessel functions of zeroth and first order

bess_J0
-------
Bessel function J0

bess_J1
-------
Bessel function J1

ell_int
-------
Complete elliptic integrals of the first and second kind

error_f
-------
Error, complementary error, and inverse error function

exp_int
-------
Exponential integral

ker_kei_0
---------
Evaluates the Kelvin functions ker_0, kei_0

ker_kei_0_large
---------
Evaluates the Kelvin functions ker_0, kei_0
for large arguments

leg_fnc_ortho
--------------
Evaluates the orthonormalized set of Legendre functions

poly_ortho
----------
Generates the coefficients of families
of orthogonal polynomials
using the Gram--Schmidt orthogonalization method
