
CHAPTER 8: Function approximation
---------------------------------

bernstein
---------
Approximates a function with a Bernstein polynomial
and plots the approximation

bsplines
---------
Non-uniform B-spline approximation

bspl_un
--------
Computes uniform B-splines for displaying purposes

fft
---
Fast Fourier Transform of a set of complex data

fourier_id
----------
Verifies the identities shown in Table 8.6.2
of Pozrikidis (1998)

ft_1d
-----
Computes the complex, cosine, and sine Fourier transform
of a function of one variable for evenly-spaced points
using the conventional method

ft_1d_nu
--------
Computes the complex Fourier transform
of a periodic function of one variable for 
non-uniformly spaced points
using the conventional method

ft_2d
-----
Computes the complex Fourier transform of a function
of two variables for evenly-spaced points
using the conventional method
